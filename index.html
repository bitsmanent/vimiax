Vimiax is a Google Chrome extension which brings the Vim interface to the browser. It allows to full control the browser actions (navigation, history, bookmarks, sessions, etc.) using the Vim key bindings; thus, without using the mouse at all.

The current implementation actually includes the following features:

  • Scrolling support
  • Navigation utilities
  • History browsing
  • Tabs management
  • Zoom facilities
  • Smart link hints
  • Incremental search
