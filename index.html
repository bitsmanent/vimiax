Vimiax Brings the Vim interface to Google Chrome

Vimiax is a Google Chrome extension which aims to port the Vim interface to the browser. It allows to full control the browser (navigation, history, bookmarks, sessions, etc.) using Vim-like key bindings; thus, without using the mouse at all.

Note: the project is a work in progress, no stable releases has been published yet but as far as I can tell the extension seems enough stable to be used by everyone. Try it out.
If you're a developer download or browse the sources in order to start using and (hopefully) contributing to Vimiax. It's needed lots of testing and at least a stub of user guide; that's mainly why it's not so easy for non-developers to try out which keycombo do what thing and it's not so easy for a developer to find the time to write a bit of documentation. So, if you are not a developer but you think to be able to understand how Vimiax works or how to use it, feel free to write something about and contacts me. You can of course add credits to yourself if you wants.

The current implementation actually includes the following features:

  • Scrolling support
  • Navigation utilities
  • History browsing
  • Tabs management
  • Zoom facilities
  • Smart link hints
  • Incremental search
Further changes will likely include sessions and bookmarks manager. We'll have to wait for the Chromium API to be extendend in many ways in order to provides a quality product merely comparable with Mozilla alternatives like Vimperator. First, it's needed to have a support for the bars which will allow us to mantain a clear and faster extension. That's why there's the Vimiax own bar implemented at the moment. Second, things like undo closed tabs or so are better implemented in-browser. So, we don't need anymore to mantain a list of all tabs (opened/closed) in order to provides such features; remember that the feature is also provided by the browser though there's no API hook for the extensions yet.
